# universal_bot-chat
Universal chat for run Alise skills or other voice assistants on the website.
- [git](https://github.com/max36895/universal_bot-chat)
- [npm](https://www.npmjs.com/package/umchat)

## Run
Install dependencies 
```bash
npm i
```
Run project
```bash
npm start
```
